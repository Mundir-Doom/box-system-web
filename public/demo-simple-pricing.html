Hub-to-Hub Only

Simple pricing based only on which hub the parcel goes from and to. No complex distance calculations or zone-based pricing.

Optional Weight

Weight is completely optional. If specified, it adds extra cost per kg. If not specified, only base hub-to-hub price applies.

Details Only

Delivery type and category are just for reference and tracking. They don't affect pricing at all - keeping it simple.

Easy Management

Beautiful interface to set pricing for each hub-to-hub combination. Set base price, per-kg rate, and min/max limits.
